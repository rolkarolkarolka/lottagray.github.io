<!DOCTYPE html>
<html lang="pl">
<head>
  <meta charset="UTF-8">
  <title>Wszechświat Rolki</title>
  <style>
    body {
      margin: 0;
      font-family: "Comic Sans MS", cursive, sans-serif;
      background: black;
      color: white;
      overflow-x: hidden;
    }

    /* Tło gwiazdek */
    .stars {
      position: fixed;
      width: 100%;
      height: 100%;
      top: 0; left: 0;
      background: black;
      overflow: hidden;
      z-index: -1;
    }

    .star {
      position: absolute;
      width: 2px;
      height: 2px;
      background: white;
      border-radius: 50%;
      opacity: 0.8;
      animation: twinkle 2s infinite alternate;
    }

    @keyframes twinkle {
      from { opacity: 0.2; transform: scale(1); }
      to   { opacity: 1; transform: scale(1.3); }
    }

    /* Nagłówek */
    header {
      background: linear-gradient(to right, #0033cc, #3366ff);
      color: #ffff99;
      text-align: center;
      font-size: 2.5em;
      padding: 15px;
      text-shadow: 0 0 8px #fff, 0 0 12px #ffff66;
      border-bottom: 4px solid yellow;
    }

    /* Układ strony */
    .container {
      display: flex;
      min-height: 100vh;
    }

    nav {
      width: 200px;
      background: #0044cc;
      padding: 15px;
      color: white;
    }

    nav a {
      display: block;
      color: white;
      text-decoration: none;
      margin: 10px 0;
      font-weight: bold;
    }

    nav a:hover {
      color: yellow;
      text-shadow: 0 0 5px #fff;
    }

    main {
      flex: 1;
      background: #fffaa0;
      color: black;
      padding: 20px;
    }

    h1, h2 {
      margin-top: 0;
    }
  </style>
</head>
<body>
  <!-- Gwiazdki w tle -->
  <div class="stars" id="stars"></div>

  <!-- Nagłówek -->
  <header>Wszechświat Rolki</header>

  <!-- Układ strony -->
  <div class="container">
    <nav>
      <a href="#">Strona Główna</a>
      <a href="#">Opowiadania</a>
      <a href="#">Wspomnienia</a>
      <a href="#">Blog</a>
      <a href="#">Galeria</a>
      <a href="#">Kontakt</a>
    </nav>

    <main>
      <h2>Witaj w moim wszechświecie!</h2>
      <p>
        To moja strona w retro stylu 🌌.  
        Tutaj znajdziesz opowiadania, przemyślenia i inne rzeczy, którymi się zajmuję.
      </p>
    </main>
  </div>

  <!-- Webneko -->
  <script>
    NekoType="valentine";
  </script>
  <h1 id="nl">
    <script src="https://webneko.net/n20171213.js"></script>
    <a href="https://webneko.net">Neko</a>
  </h1>

  <!-- Skrypt do generowania gwiazdek -->
  <script>
    const starsContainer = document.getElementById("stars");
    for (let i = 0; i < 100; i++) {
      const star = document.createElement("div");
      star.className = "star";
      star.style.top = Math.random() * 100 + "%";
      star.style.left = Math.random() * 100 + "%";
      star.style.animationDuration = (1 + Math.random() * 3) + "s";
      starsContainer.appendChild(star);
    }
  </script>
</body>
</html>

ChatGPT said:

Super pytanie 👍
Pokażę Ci krok po kroku, jak wrzucić Twoją stronę (index.html + podstrony) na GitHub Pages tak, żeby była dostępna pod adresem np. twojanazwa.github.io.

🔹 Krok 1: Utwórz repozytorium

  1. Zaloguj się na GitHub.

  2. Kliknij New repository (nowe repozytorium).

  3. Nazwij repozytorium np. wszechswiat-rolki.

  4. Zaznacz „Public” (musi być publiczne, żeby GitHub Pages działało).

  5. Kliknij Create repository.

🔹 Krok 2: Wgraj pliki

  1. Na stronie repozytorium kliknij Add file → Upload files.

  2. Wgraj wszystkie pliki swojej strony:

    • index.html

    • opowiadania.html

    • wspomnienia.html

    • blog.html

    • galeria.html

    • kontakt.html

    • style.css (jeśli wydzielisz style)

    • script.js (jeśli wydzielisz skrypt do gwiazdek)

  3. Kliknij Commit changes (zatwierdź zmiany).

🔹 Krok 3: Włącz GitHub Pages

  1. Wejdź w ustawienia repozytorium (Settings).

  2. Z menu po lewej wybierz Pages.

  3. W sekcji Source wybierz:

    • Branch: main

    • Folder: / (root)

  4. Kliknij Save.

🔹 Krok 4: Otwórz swoją stronę 🌌

  • GitHub wygeneruje link do Twojej strony w sekcji GitHub Pages.

  • Adres będzie wyglądał tak:

    arduino
    https://twojanazwa.github.io/wszechswiat-rolki/

  • Po kilku minutach strona powinna działać! 🎉
