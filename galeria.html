Poka Ci krok po kroku, jak wrzuci Twoj stron (index.html + podstrony) na GitHub Pages tak, eby bya dostpna pod adresem np. twojanazwa.github.io.

Krok 1: Utwrz repozytorium

  1. Zaloguj si na GitHub.

  2. Kliknij New repository (nowe repozytorium).

  3. Nazwij repozytorium np. wszechswiat-rolki.

  4. Zaznacz Public (musi by publiczne, eby GitHub Pages dziaao).

  5. Kliknij Create repository.

Krok 2: Wgraj pliki

  1. Na stronie repozytorium kliknij Add file Upload files.

  2. Wgraj wszystkie pliki swojej strony:

    • index.html

    • opowiadania.html

    • wspomnienia.html

    • blog.html

    • galeria.html

    • kontakt.html

    • style.css (jeli wydzielisz style)

    • script.js (jeli wydzielisz skrypt do gwiazdek)

  3. Kliknij Commit changes (zatwierd zmiany).

Krok 3: Wcz GitHub Pages

  1. Wejd w ustawienia repozytorium (Settings).

  2. Z menu po lewej wybierz Pages.

  3. W sekcji Source wybierz:

    • Branch: main

    • Folder: / (root)

  4. Kliknij Save.

Krok 4: Otwrz swoj stron

  • GitHub wygeneruje link do Twojej strony w sekcji GitHub Pages.

  • Adres bdzie wyglda tak:

    arduino
    https://twojanazwa.github.io/wszechswiat-rolki/

  • Po kilku minutach strona powinna dziaa!
